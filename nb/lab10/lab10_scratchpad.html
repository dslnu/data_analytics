Postgres for analytics

Installation

Fetch the install from: https://www.postgresql.org/download/windows/

For python: pip install "psycopg[binary,pool]"

Dataset

We’ll use a shrunken version of this dataset: https://ual.sg/project/global-streetscapes/

head -n1 gadm.csv > gadm_sorted.csv
tail -n+2 gadm.csv | sort -k1 -t, >> gadm_sorted.csv
head -n10001 gadm_sorted.csv > ../../gadm.csv

Usage

First, create a database in your cluster:

 create database lab10 owner <your_username>;
